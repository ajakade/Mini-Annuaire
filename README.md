# Mini-Annuaire React

## Description

Mini-Annuaire est une application web développée avec **React** permettant de gérer et afficher une liste d'utilisateurs.  
Elle inclut les fonctionnalités suivantes :  

- Affichage d'une table des utilisateurs (nom, ville, entreprise, email).  
- rechecher d'utilisateur
- Afficahe des details 
- Mode sombre activable pour l’ensemble de l’application.  

Cette application est conçue pour être simple, intuitive et responsive.

---

## Technologies utilisées

- **React** (v18)  
- **JavaScript (ES6)**  
- **CSS** pour le style et le mode sombre
